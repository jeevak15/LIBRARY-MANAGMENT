# 📚 Library Management System in C

A simple and professional **Library Management System** developed using the **C Programming Language** for engineering mini-project submission.

This project demonstrates core C programming concepts like:
- Structures
- Functions
- Arrays
- File Handling
- Searching
- Sorting
- Menu-Driven Programming

---

# 🚀 Features

✅ Add New Books  
✅ Display All Books  
✅ Search Books  
✅ Delete Books  
✅ Update Book Records  
✅ Issue Books  
✅ Return Books  
✅ File Storage System  
✅ User-Friendly Menu Interface

# ⚙️ How to Run

## Using GCC Compiler

```bash
gcc main.c -o library
./library
```

## Using Turbo C

1. Open Turbo C
2. Open `main.c`
3. Compile and Run
